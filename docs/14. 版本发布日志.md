# 版本发布日志

## 0.1.0 (2026-02-24)

### 新增功能
- ✅ 核心功能框架
- ✅ 配置管理模块
  - 配置文件加载和保存
  - 原子保存防止文件损坏
  - 配置验证
  - 向后兼容处理
- ✅ 环境变量管理模块
  - 读取/设置/删除环境变量
  - PATH 环境变量管理
  - 环境变量变更广播
- ✅ 版本管理模块
  - 扫描本地已安装版本
  - 获取远程可用版本
  - 下载并安装版本
  - 卸载版本
  - 切换版本
- ✅ 下载管理模块
  - 多镜像源支持
  - 下载进度回调
  - 下载重试机制
  - 下载速度限制
  - 请求频率限制
- ✅ 缓存管理
  - 远程版本列表缓存
  - 缓存过期时间配置
- ✅ GUI 界面
  - 主窗口
  - 侧边栏导航
  - 版本管理面板
  - 配置管理面板
  - 各种对话框
- ✅ CLI 命令行接口
  - list 命令：列出版本
  - use 命令：切换版本
  - install 命令：安装版本
  - uninstall 命令：卸载版本
  - root 命令：设置根目录
  - config 命令：管理配置
  - tools 命令：列出工具
- ✅ 工具类
  - 日志系统
  - 权限管理
  - 输入验证
  - 重试装饰器
  - 频率限制器
  - 速度限制器

### 技术栈

- QML

### 预配置工具
- Python（官方源、华为云、阿里云镜像）

## 版本说明

### 版本号规则
采用语义化版本（Semantic Versioning）：
- MAJOR：重大变更，不兼容的 API 修改
- MINOR：向下兼容的功能性新增
- PATCH：向下兼容的问题修正

### 发布周期
- 不定期发布，根据功能完成情况
