# 开发环境搭建指南

## 1. 概述

本文档描述如何搭建 Mysysenv 项目的开发环境。

## 2. 系统要求

### 2.1 操作系统
- Windows 7 或更高版本

### 2.2 软件要求
- Python 3.14 或更高版本
- Git（可选，用于版本控制）

## 3. 环境搭建步骤

### 3.1 克隆或下载项目

#### 方式一：使用 Git 克隆
```bash
git clone <repository-url>
cd mysysenv
```

#### 方式二：直接下载
下载项目 ZIP 包并解压。

### 3.2 创建虚拟环境（推荐）

```bash
# 在项目根目录下执行
python -m venv .venv

# 激活虚拟环境
# Windows PowerShell
.venv\Scripts\Activate.ps1

# Windows CMD
.venv\Scripts\activate.bat
```

### 3.3 安装依赖

```bash
pip install PySide6==6.8
```

### 3.4 验证环境

```bash
# 检查 Python 版本
python --version

# 检查 PySide6 版本
python -c "import PySide6; print(PySide6.__version__)"
```

## 4. 运行项目

### 4.1 运行 GUI 模式

```bash
python src/main.py
```

### 4.2 运行 CLI 模式

```bash
# 查看帮助
python src/main.py --help

# 列出版本
python src/main.py list python

# 其他命令...
```

## 5. 开发工具推荐

### 5.1 IDE/编辑器
- PyCharm（推荐）
- VS Code
- 其他支持 Python 的 IDE

### 5.2 VS Code 扩展推荐
- Python
- Pylance
- QML Syntax Highlighting
- GitLens

### 5.3 PyCharm 配置
1. 打开项目
2. File → Settings → Project: mysysenv → Python Interpreter
3. 选择 .venv 虚拟环境
4. 配置 Run/Debug Configuration

## 7. 常见问题

### 7.1 虚拟环境激活失败

**问题**：PowerShell 中无法激活虚拟环境

**解决**：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 7.2 PySide6 安装失败

**问题**：pip install PySide6 失败

**解决**：
- 检查网络连接
- 使用国内镜像源：
```bash
pip install PySide6==6.8 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 7.3 权限问题

**问题**：运行时提示需要管理员权限

**解决**：
- 以管理员身份运行终端
- 或使用 GUI 模式，会自动请求权限
