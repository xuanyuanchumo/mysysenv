# 原型图

## 1. 概述

本文档描述 Mysysenv 应用的用户界面原型设计。

## 2. 主界面布局

### 2.1 整体结构
```
┌─────────────────────────────────────────────────────────────┐
│ Mysysenv - 系统环境管理器                    [关闭][最小化] │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  侧边栏  │              主内容区域                          │
│          │                                                  │
│  Python  │                                                  │
│  Java    │                                                  │
│  Node    │                                                  │
│  Maven   │                                                  │
│  Gradle  │                                                  │
│          │                                                  │
│ [+ 添加] │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 2.2 侧边栏
- 显示已配置的工具列表
- 当前选中的工具高亮显示
- 底部有"添加工具"按钮

│  已安装版本:                              │

## 6. CLI 界面示例

### 6.1 帮助信息
```
$ mse --help
usage: mse [-h] [--version] [--verbose] [--config CONFIG]
           {list,use,install,uninstall,root,config,tools} ...

Mysysenv - 系统环境管理器

options:
  -h, --help            显示帮助信息
  --version             显示版本号
  -v, --verbose         启用详细输出
  -c CONFIG, --config CONFIG
                        配置文件路径

命令:
  {list,use,install,uninstall,root,config,tools}
    list                列出工具的已安装版本
    use                 切换到指定版本
    install             下载并安装指定版本
    uninstall           卸载指定版本
    root                设置或显示工具根目录
    config              显示或编辑配置
    tools               列出所有已配置的工具
```

### 6.2 列出版本示例
```
$ mse list python
python 已安装版本:
 * 3.11.4
   3.10.12
   3.9.13

当前版本: 3.11.4
```

### 6.3 切换版本示例
```
$ mse use python 3.10.12
正在切换 python 到版本 3.10.12...
成功切换 python 到 3.10.12
注意：可能需要重启终端才能使更改生效。
```
